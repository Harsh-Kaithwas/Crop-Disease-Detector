An intelligent Android application designed to assist farmers and agricultural experts. The app uses machine learning to identify crop diseases instantly from images and provides a chatbot-style guidance interface.

---

## 🚀 Key Features

- **📸 Smart Disease Detection:** Analyzes leaves using a custom TFLite model.
- **💬 Expert Mode Chatbot:** Provides tailored advice and treatments for different crop diseases.
- **⚡ Offline Processing:** Runs locally on the device with no internet requirement.
- **📱 Simple & Clean UI:** Easy-to-use interface for non-technical users.

---

## 🛠️ Tech Stack

- **Development Environment:** Android Studio
- **Language:** Java
- **Machine Learning:** TensorFlow Lite Model Inference
- **UI & Layouts:** XML
