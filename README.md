🌿 Crop Disease Detector
📌 About the Project
An intelligent Android application designed to assist farmers and agricultural experts. The app uses Machine Learning (TensorFlow Lite) to identify crop diseases instantly from images and provides a chatbot-style guidance interface to help users manage plant health.

🚀 Key Features
Smart Disease Detection: Analyzes leaves using a custom-trained TFLite model to identify health status and diseases.

Expert Mode: Provides tailored advice and treatments for different crop diseases.

Offline Processing: Runs the machine learning model directly on the device without requiring an active internet connection for detection.

Simple & Clean UI: Easy-to-use interface for farmers and non-technical users.

🛠️ Tech Stack
Language: Java

Machine Learning: TensorFlow Lite

Development Environment: Android Studio

Layouts: XML
